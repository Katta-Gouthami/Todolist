# ✅ TaskFlow - To-Do App

A clean, minimalist task manager built with vanilla JavaScript. Add tasks, mark them complete, and delete them - all saved in your browser.

## 🚀 Live Demo
**[Try TaskFlow Live](https://katta-gouthami.github.io/Todolist/
)**

## ✨ Features

- ✅ **Add Tasks** - Type and press Enter or click Add
- ✅ **Complete Tasks** - Check the box to mark done
- ✅ **Delete Tasks** - Remove tasks you don't need
- ✅ **Persistent Storage** - Tasks save automatically in your browser
- ✅ **Task Counter** - See total tasks at a glance
- ✅ **Keyboard Support** - Press Enter to add quickly
- ✅ **Responsive Design** - Works on mobile, tablet, and desktop

## 🛠️ Built With

- HTML5
- CSS3 (Flexbox, Grid, Custom Properties)
- JavaScript (ES6)
- localStorage API
